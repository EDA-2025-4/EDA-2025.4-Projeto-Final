# EDA-2025.4-Projeto-Final

## Objetivo

O objetivo deste trabalho é desenvolver um sistema na linguagem **C** que utilize **listas simplesmente encadeadas** para armazenar e manipular os dados dinamicamente, implementando operações completas de **CRUD**.

##  Funcionalidades

*   Gerenciamento de Clientes;
*   Geremciamento de Produtos;
*   Modo Compra;

---

##  Clientes 

*   O Sistema Permite Que Os Clientes Se Cadastrem de Forma Que Uilize o (CPF, nome, email, telefone, data de nascimento);
*   Mostrar Todos os Clientes Listados;
*   Buscar cliente pelo CPF;Buscar cliente pelo CPF;
*   Editar dados de um cliente;
*   Remover cliente;

---

##  Produtos 

*   O Sistema Permite Que Os Produtos Sejam Cadastrados como (código único, nome ,preço, quantidade);
*   Mostrar Todos Os Produtos Disponiveis;
*   Buscar Produto Pelo Código;
*   Editar Dados De Um Produto;
*   Remover Produto;

---

##  Modo Compra

*   O Sistema Permite Que O Cliente Inclua Produtos Que Deseja Comprar Ao Carrinho;
*   Permite Listar Os Produtos Do Carrinho De Um Cliente e Informar Quantidade De Itens e Valor Total Da Compra;
*   Permite Retirar os produtos do carrinho de um cliente;
